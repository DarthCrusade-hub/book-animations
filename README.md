# Book Animations

Blender animation project for a book series.

## Folder Structure

```
book-animations/
├── main.blend          # Main project file (generated by init script)
├── scenes/             # Per-chapter or per-episode scene files
├── characters/         # Character rigs and models
├── assets/             # Reusable props, environments, materials
├── renders/            # Render output (gitignored)
├── scripts/            # Python utilities for Blender
└── config/             # Project configuration
```

## Setup

1. Install [Blender](https://www.blender.org/download/) 4.0 or newer.
2. Initialize the main project file:

```powershell
blender --python scripts/init_project.py
```

3. Open the project:

```powershell
blender main.blend
```

## Scripts

| Script | Purpose |
|--------|---------|
| `scripts/init_project.py` | Create `main.blend` with starter scene and collections |
| `scripts/project_paths.py` | Path helpers and directory setup |
| `scripts/render_scene.py` | Background rendering |
| `scripts/link_asset.py` | Append/link assets from `assets/` |

### Examples

```powershell
# Verify paths
python scripts/project_paths.py

# Render a still frame in the background
blender main.blend --background --python scripts/render_scene.py

# Render an animation to renders/preview/
blender main.blend --background --python scripts/render_scene.py -- --output renders/preview --animation --frame-start 1 --frame-end 120

# Link a prop from the asset library
blender main.blend --python scripts/link_asset.py -- --file assets/props/book.blend --object Book --link
```

## Workflow

- Save scene-specific work under `scenes/` (e.g. `scenes/chapter_01.blend`).
- Keep character rigs in `characters/` and reference them via linking.
- Store reusable props and sets in `assets/`.
- Configure defaults in `config/project.json`.
